# Portfolio - Vicky Kumar

This is a personal portfolio website for Vicky Kumar. It is a static site built with HTML, CSS and JavaScript.

## Structure
- `index.html` — main page
- `style.css` — styles
- `script.js` — client-side behaviors
- `images/` — project and profile images

## Local development
Open `index.html` in a browser or use a local static server. Example (PowerShell):

```powershell
# simple static server using Python 3
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

## Notes
- Make sure to keep any sensitive keys out of the repository (use `.env` and add it to `.gitignore`).
